# AA12 - Sistema de Cadastro com CEP e Localidade

1. CSS e estrutura do AA8 (navegação, formulários e validações)
2. Funcionalidades do AA12 (CEP e localidade)

## Arquivos
- `index.html`: Página principal com os formulários e navegação
- `css/style.css`: Estilos do AA8 + novos estilos para AA12
- `js/script.js`: Script principal com validações e funções
- `js/dropdown.js`: Script para o menu dropdown
- `js/password_visibility.js`: Script para toggle de visibilidade de senha

## Funcionalidades

### AA8 (base)
- Menu de navegação (Home, Login, Cadastro)
- Validação de formulários
- Visibilidade de senha
- Feedback visual de validação

### AA12 (novas)
- Integração com ViaCEP
- Integração com IBGE Localidades
- Autopreenchimento de endereço
- Seletores dinâmicos de UF/Município
