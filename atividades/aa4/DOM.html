📖 Sobre o DOM

O Document Object Model (DOM) é uma representação estrutural do documento HTML como uma árvore de objetos.

🏗️ HTML Semântico

O projeto utiliza tags semânticas para melhor estruturação e acessibilidade:

  • <header> - Cabeçalho
  • <nav> - Navegação
  • <main> - Conteúdo principal
  • <aside> - Barra lateral
  • <section> - Seções
  • <article> - Artigos
  • <footer> - Rodapé
  • <address> - Contato

  • Melhor SEO

Tags Semânticas Utilizadas

Análise das tags HTML5 semânticas implementadas no projeto

<html> - Elemento raiz do documento
<head> - Metadados do documento
<body> - Corpo do documento
<header> - Cabeçalho principal com navegação
<nav> - Menu de navegação principal
<div.main-container> - Container principal
<aside> - Barra lateral com informações relevantes
<main> - Conteúdo principal da página
<section> - Seções temáticas (sobre, objetivos, disciplina, game, rules)
<header> - Cabeçalho de cada seção
<article> - Artigos independentes dentro das seções
<footer> - Rodapé com informações e links
<address> - Informações de contato do grupo

🏷️ Tags Semânticas Principais

  • <header> - Define o cabeçalho da página com navegação
  • <nav> - Contém os links de navegação principais
  • <main> - Delimita o conteúdo principal da página
  • <aside> - Barra lateral com informações complementares
  • <section> - Agrupa conteúdo temático relacionado
  • <article> - Conteúdo independente e reutilizável
  • <footer> - Rodapé com informações e links úteis
  • <address> - Informações de contato específicas

✅ Benefícios da Estrutura Semântica

  • SEO Melhorado: Motores de busca compreendem melhor o conteúdo
  • Acessibilidade: Leitores de tela navegam mais facilmente
  • Manutenibilidade: Código mais legível e organizados
  • Padrões Web: Segue as melhores práticas do HTML5
  • Responsividade: Facilita o design responsivo
  • Performance: Estrutura otimizada para carregamento

🎯 Atributos e Metadados

  • lang="pt-BR": Define o idioma como Português Brasileiro
  • charset="UTF-8": Codificação de caracteres Unicode
  • viewport: Configuração para dispositivos móveis
  • meta description: Descrição para motores de busca
  • meta keywords: Palavras-chave relevantes
  • favicon.ico: Ícone da aba do navegador
  • ARIA labels: Melhor acessibilidade
